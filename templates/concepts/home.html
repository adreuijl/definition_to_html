Welkom bij de SEMA definition to html app

Hier kan je in een aantal stappen je definities matchen tegen een in te stellen endpoint/database


De stappen zijn als volgt:

  • Stel het endpoint in van de bron waartegen definities gematched gaan worden Set endpoint
  • Upload je ttl file van begrippen uit jouw nieuw gemaakte begrippenkader Upload turtle file
  • Klik op de button: Maak Match
  • Bekijk de resultaten van de match en pas deze waar-nodig handmatig aan, exporteer ze daarna om ze te uploaden naar EDG Check Concepts
